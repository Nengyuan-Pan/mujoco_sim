# Exp11 — Stage -1 回归测试：力矩 vs 位置模式全面对比

**日期**: 2026-06-13  
**实验编号**: exp11_regression  
**数据目录**: `experiment_data/exp11_regression/`  
**CSV**: `results.csv` (1000 行)  
**运行时间**: 3.4 min (8 workers), 0 failed  

## 目的

验证 Stage -1 位置控制模式（`--position-mode`, Kp=20, Kd=1.6 uniform）在不同球速和噪声条件下，与力矩模式相比的性能表现。验收标准：位置模式命中率 ≥ 力矩模式 × 90%，位置误差 ≤ 力矩模式 × 1.5。

## 参数

| 参数 | 值 |
|------|-----|
| 球速 | 6, 7, 8, 9, 10 m/s (5 级) |
| 模式 | torque (力矩), position (位置 `--position-mode`) |
| 噪声 | off (无噪声), lo (σ_p=0.02, σ_v=0.2) |
| 位置模式 PD | Kp=[20]×6, Kd=[1.6]×6 (uniform, exp10 精调选定) |
| Seeds | 0-49 (50 个) |
| 发球模式 | serve_box |
| 脚本 | V11 (`rm65_mpc_v11.py`) |
| 总运行数 | 5 × 2 × 2 × 50 = 1000 |

> **注意**: `--obs-noise-pos/vel` 参数需配合 `--obs-freq` 使用（BallObservationGate 门控），单独使用不生效。本次实验 lo 噪声条件实际等同 off 噪声，噪声鲁棒性对比留待后续。

## 结论

### 数据观察（Agent 生成）

1. **位置模式通过 Stage -1 验收**：在 7m/s 标准球速下，位置模式命中率 90%（vs 力矩 94%），比率 95.7% ≥ 90% 验收线。位置误差 54.1mm（vs 力矩 40.2mm），比值 1.35x ≤ 1.5 验收线。位置模式 PD 控制器（Kp=20, Kd=1.6 uniform）成功达到力矩模式 95.7% 的性能水平。

2. **位置模式在高球速下显著优于力矩模式**：9m/s 下位置模式命中率 38% vs 力矩 22%（+16pp，比率 173%），10m/s 下 26% vs 14%（+12pp，比率 186%）。根因：位置模式 TCP 峰值速度 2.36-2.41 m/s，远高于力矩模式的 1.69-1.73 m/s。低 Kp（20）使 PD 控制器更柔顺，安全滤波器更少触发速度限制，允许更大挥拍行程。这一发现表明位置控制模式在高动态场景下可能比力矩控制更有优势。

3. **位置模式位置误差在高球速下更低**：8-10m/s 下位置模式误差（71-80mm）均低于力矩模式（80-98mm），误差比 0.80-0.89x（< 1.0 意味着位置模式更精确）。这与 7m/s 下位置模式误差更高（1.35x）形成对比——低球速时力矩模式精确跟踪更优，高球速时位置模式柔顺性带来更好的末端到达。

4. **噪声注入未生效**：lo 噪声与 off 噪声结果完全一致（退化 0pp），因为 `--obs-noise-pos/vel` 参数需要配合 `--obs-freq`（BallObservationGate 门控）才能激活。本次实验 1000 runs 中 lo 条件实际等同 off。噪声鲁棒性对比需后续补充 `--obs-freq 200`。
