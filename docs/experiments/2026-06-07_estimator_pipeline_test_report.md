## 1. 测试目的

验证感知模块三层架构的端到端可靠性：

```
仿真层(MuJoCo真值) → 实验层(add_observation_noise) → 感知层(BallEstimator KF) → 输出
```

核心验证点：
1. 噪声是否正常注入（raw RMSE ≈ 设定 σ）
2. 滤波是否有效衰减噪声（filt RMSE < raw RMSE × 阈值）
3. 弹跳场景下滤波器是否正常跟踪
4. 高噪声下是否数值稳定（无 NaN/Inf）
5. RM65Env 全链路数据通路是否通畅

**注意**：本测试不涉及机器人规划/控制，仅验证噪声→滤波 pipeline。

## 2. 测试架构

### 数据流

```
┌─────────────┐     ┌──────────────────┐     ┌─────────────────┐
│  MuJoCo     │     │ add_observation  │     │  BallEstimator  │
│  物理引擎   │────→│ _noise()         │────→│  6D KF 滤波     │────→ 滤波输出
│  (真值)     │     │ (高斯噪声)       │     │  (predict+      │
└─────────────┘     └──────────────────┘     │   correct)      │
       │                                     └─────────────────┘
       │                                              │
       └────── 直接读 data.qpos/qvel ────────────────┘
                          (作为对比基准)
```

### 每帧循环（MuJoCo 用例 1-4）

```python
for i in range(n_steps):
    # 1. MuJoCo 真值（绕过 estimator，直接读物理引擎）
    true_pos = env.data.qpos[bq:bq+3].copy()
    true_vel = env.data.qvel[bv:bv+3].copy()
    # 2. 加噪声
    noisy_pos, noisy_vel = add_observation_noise(true_pos, true_vel, rng, ...)
    # 3. KF 滤波
    filt_pos, filt_vel = estimator.update(noisy_pos, noisy_vel)
    # 4. 推进物理（机器人不动，仅球飞行）
    env.step(np.zeros(6))
```

## 5. 关键发现

1. **滤波效果稳健**：所有场景位置 RMSE 衰减比在 0.17-0.21 之间，速度在 0.22-0.32 之间，不随噪声等级显著变化。

2. **弹跳处理有效**：弹跳后 20 步内滤波 vz 跟踪真实 vz（偏差 0.012 m/s），slam 保护机制工作正常。

3. **高噪声不发散**：σ_p=10cm（球半径 3 倍）下仍数值稳定，终点误差仅 1.1cm。

4. **各向异性正确处理**：per-axis R 矩阵按预期工作，深度方向大噪声被正确建模。

5. **噪声注入与 exp7 对齐**：lo/mid/hi/anis 参数与 exp7 实验等级一致，结论可直接迁移到实验分析。
