# 实验记录：exp1 算法能力上限（速度豁免，bounce 模式）

**日期**: 2026-06-03
**实验编号**: exp1_algorithm_capability
**数据目录**: `experiment_data/exp1_algorithm_capability/`

## 目的

速度豁免模式下（forward_pass_margin=3.0），验证算法在 bounce 发球时的击打能力上限（8-18 m/s）。

## 参数

| 变量 | 值 |
|------|-----|
| 球速范围 | 8-18 m/s, 1m/s 粒度 |
| Tube | on/off |
| Seeds | 0-9（10 个） |
| 约束 | 豁免（forward_pass_margin=3.0, qdot_scale=0.95, max_tcp=∞） |
| 脚本 | `rm65_mpc_tube_constraint.py`（离线） |
| 发球模式 | serve_box bounce |
| 总运行数 | 220 |

## 结果

| 球速 | Tube ON | Tube OFF | 误差 ON/OFF | active |
|------|---------|----------|-------------|--------|
| 8 m/s | 8/10 | 8/10 | 52/30mm | 0 |

| 18 m/s | 4/10 | 4/10 | 94/89mm | 0 |

## 结论

1. 算法上限 ~15 m/s（85%命中），16-18 m/s 断崖式下降到 40%。
2. 全部 220 次命中均为被动接触（active=0），说明瓶颈在代价函数速度跟踪，非关速约束。
3. Tube ON/OFF 无显著差异。
